> Uppercase the first character in the string.

## :cloud: Installation

```sh
$ npm i --save uc-first
```


## :clipboard: Example



```js
const ucFirst = require("uc-first");

console.log(ucFirst("hello world!"));
// => Hello world!
```

## :memo: Documentation


### `ucFirst(a, b)`
Uppercase the first character in the string.

#### Params
- **Number** `a`: Param descrpition.
- **Number** `b`: Param descrpition.

#### Return
- **Number** Return description.
